**ProgAce** is a high-fidelity career mastery platform designed to bridge the gap between Junior Developers and Staff-level Architects. Built with **Adaptive AI**, it transforms the standard "tutorial hell" into a data-driven journey of professional excellence.

---

## 🧭 Mastery Workflow

The ProgAce experience is built on a 4-step progressive track designed to ensure maximum retention and interview success.

![ProgAce Mastery Workflow](./public/workflow.png)

1.  **Diagnostic Suite**: AI-powered analysis of your current technical baseline across DSA, System Design, and CS Fundamentals.
2.  **Adaptive Blueprint**: A 100% personalized roadmap generated by Claude 3.5, optimized for your target company (Google, Meta, Stripe) and timeline.
3.  **Elite Training**: Mastery-based learning modules with gatekeeper tests, ensuring you don't move forward until you've truly mastered the topic.
4.  **Staff-Level Interview**: AI Voice Mock Interviews that simulate real-world technical bar-raising sessions.

## 💎 Core Features

### 🏢 Expert Industry Paths
Direct access to 180+ expert-verified roadmaps for specific roles at top-tier tech firms.
*   **Company Specific**: Targeting L5 SDE at Google? We have the exact blueprint.
*   **Role Focused**: Specialized tracks for Frontend, Backend, ML, and AI Engineers.

### 🧩 Strategic Nexus (Roadmap)
A high-fidelity control center for your career journey.
*   **SWOT Analysis**: Real-time identification of Strengths, Weaknesses, Opportunities, and Threats.
*   **Detailed Timeline**: Granular daily tasks mapped to your commitment level (2-8 hrs/day).

### 🎙️ AI Mock Interviewer
Practice like you play. Our AI voice assistant conducts real-time technical interviews.
*   **Voice-Native**: Communicate naturally; the AI adapts its follow-up questions based on your responses.
*   **Instant Feedback**: Get a diagnostic score on your communication, logic, and coding performance.

### ⚔️ Code Arena
A premium, integrated workspace for competitive programming and system design practice.
*   **Multi-Model Intelligence**: Get hints and code reviews from Claude 3.5 and Llama 3.3.
*   **Production Standards**: Learn to write code that meets Staff-level architectural standards.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Set up your environment variables in `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
GROQ_API_KEY=your-groq-api-key
ANTHROPIC_API_KEY=your-anthropic-api-key
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to witness your career transformation.
