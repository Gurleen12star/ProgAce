**ProgAce** is a high-fidelity career mastery platform designed to bridge the gap between Junior Developers and Staff-level Architects. Built with **Adaptive AI**, it transforms the standard "tutorial hell" into a data-driven journey of professional excellence.

---

## 🧭 Mastery Workflow

The ProgAce experience is built on a 4-step progressive track designed to ensure maximum retention and interview success.


1.  **Diagnostic Suite**: AI-powered analysis of your current technical baseline across DSA, System Design, and CS Fundamentals.
2.  **Adaptive Blueprint**: A 100% personalized roadmap generated by Claude 3.5, optimized for your target company (Google, Meta, Stripe) and timeline.
3.  **Elite Training**: Mastery-based learning modules with gatekeeper tests, ensuring you don't move forward until you've truly mastered the topic.
4.  **Staff-Level Interview**: AI Voice Mock Interviews that simulate real-world technical bar-raising sessions.

   ```mermaid
flowchart TD
    A[User Developer] --> B[Diagnostic Suite]

    B --> C[Analyze Skills DSA System Design CS Fundamentals]

    C --> D[Adaptive Blueprint AI Generated Roadmap]

    D --> E[Personalized Plan Target Company Timeline]

    E --> F[Elite Training Modules]

    F --> G{Mastery Check}

    G -->|Pass| H[Next Module]
    G -->|Fail| F

    H --> I[AI Mock Interview Voice Based]

    I --> J[Feedback Scoring Communication Logic Coding]

    J --> K[Career Ready Staff Level]
```

## 🛠️ Technology Stack

*   **Frontend**: Next.js 15 (App Router), CSS Modules, Lucide Icons.
*   **Backend**: Supabase (PostgreSQL, Auth, RLS).
*   **Intelligence**: 
    *   **Claude 3.5 Sonnet** (Primary Architect & Strategic Advice).
    *   **Llama 3.3 70B** (via Groq for low-latency code execution analysis).
*   **Styling**: Vanilla CSS with Glassmorphism and Neon-Gradient effects.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Set up your environment variables in `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
GROQ_API_KEY=your-groq-api-key
ANTHROPIC_API_KEY=your-anthropic-api-key
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to witness your career transformation.
